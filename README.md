## 🔑 Setup

1. Get a free Gemini API key from [aistudio.google.com](https://aistudio.google.com)
2. Open the app and click **"Demo Mode — Click to Connect AI"**
3. Enter your key → **Activate Intelligence**

> **Free tier:** 1,500 requests/day — more than enough for personal use.

## 🛠 Tech Stack

- **Vanilla HTML/CSS/JS** — zero dependencies, zero build step
- **Google Gemini 2.0 Flash** — AI backend
- **Three.js** — 3D particle background
- **Chart.js** — Radar + doughnut charts
- **Web Audio API** — Ambient soundscape
- **html2canvas** — Identity card generation
